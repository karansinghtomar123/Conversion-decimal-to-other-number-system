Decimal to Binary

Algorithm

for(let i=0;number!=0;i++)
{
digit[i]=number%2;
number=Math.floor(number/2);
}

Example

Divisor Quotient Remainder
2 10 0
2 5 1
2 2 0
1 1
Output is: (1010)2

Decimal to Hexadecimal

Algorithm

for(let i =0;number!=0;i++)
{
var temp = 0;
temp = number % 16;
if(temp < 10)
{ digit[i] = temp ;}
else if(temp==10 )
{ digit.push("A");}
else if(temp==11 )
{ digit.push("B");}
else if(temp==12 )
{ digit.push("C");}
else if(temp==13 )
{ digit.push("D");}
else if(temp==14 )
{ digit.push("E");}
else if(temp=15 )
{ digit.push("F");}
number=Math.floor(number/16);
}

Example

Divisor Quotient Remainder
16 155 B
9 9
Output is: (9B)16
